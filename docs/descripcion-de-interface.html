Fieldbook application user manual

2 Descripción de Interface

La aplicación se separa en itms de análisis estadísticos:

2.1 FieldBook

Aqui podrás contruir tu libro de campo, es decir, exportar el único formato que entiende la computadora para que después puedas cargar la data.

2.2 Importdata

Una vez recogido los datos en el formato antes exportados, ahora aquí los cargaremos a la plataforma y podrás usar la herramienta fieltro para elegir los factores y variables.

2.3 Outliers

Seguidamente, podrás dar una vista rápida a la distribución y sismetria de tus datos mediante la construcción de diagramas de cajas o boxplot.

2.4 Multivariante

Aqui se presentan pruebas de correlación, analisis de componentes principales y correlación de pearson, para su descarga

2.5 Regression

Se contruyen modelos de regresión lineal, visualización de la ecuación y representación gráfica, para su exportación

2.6 Stadistics

Se desarrollan el análisis de varianza y prueba de comparación de medias mostrada en cuadros resúmen sensible a ser exportadas.

2.7 Graphics

La visualización de los resultados en gráfico de barras o líneas en b/n y colores.
